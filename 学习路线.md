class RuleBasedAI:
    """基于规则的AI"""
    
    def choose_action(self, game_state, legal_actions):
        """
        简单策略：
        1. 如果是得分组且能拿分，优先拿分
        2. 用最小能赢的牌赢
        3. 赢不了就出最小的牌
        """
        my_role = self.get_role(game_state)
        my_hand = game_state.current_player_hand()
        
        # 策略1：能赢就赢（特别是有分时）
        if self.can_win_round(game_state, legal_actions):
            round_score = self.count_round_score(game_state)
            if round_score > 0 or my_role == 'runner':
                return self.choose_smallest_winner(legal_actions)
        
        # 策略2：赢不了就出小牌

## 📚 阶段3：MCTS（第一个真正的AI算法）

**目标**：实现MCTS，这是你学到的第一个搜索类AI

**实践知识**：
- 蒙特卡洛树搜索原理
- 搜索树数据结构
- 游戏状态复制和管理

**这是最适合你的入门AI算法！** 原因：
1. ✅ 概念直观（模拟→统计→选择）
2. ✅ 不需要训练数据
3. ✅ 不需要神经网络
4. ✅ 效果明显（肯定比规则AI强）

```python
class Node:
    """MCTS树节点"""
    def __init__(self, state, parent=None):
        self.state = state  # 游戏状态
        self.parent = parent
        self.children = {}  # {action: Node}
        self.visit_count = 0
        self.total_reward = 0

class MCTS:
    def __init__(self, iterations=100):
        self.iterations = iterations
    
    def search(self, root_state):
        """主搜索循环"""
        root = Node(root_state)
        
        for _ in range(self.iterations):
            node = self.select(root)  # 1. 选择
            self.expand(node)          # 2. 扩展
            reward = self.simulate(node)  # 3. 模拟
            self.backpropagate(node, reward)  # 4. 回溯
        
        return self.best_action(root)
    
    def select(self, node):
        """用UCB公式选择最有潜力的节点"""
        while node.is_fully_expanded():
            node = self.best_child(node, exploration_weight=1.4)
        return node
    
    def simulate(self, node):
        """随机模拟到游戏结束，返回胜负"""
        state = node.state.clone()
        while not state.is_terminal():
            action = random.choice(state.get_legal_actions())
            state.apply(action)
        return state.get_reward()
```

**UCB公式（核心！）**：
$$
UCT(v_i, v) = \frac{Q(v_i)}{N(v_i)} + c \cdot \sqrt{\frac{\ln N(v)}{N(v_i)}}
$$

- $Q(v_i)$ = 该节点累计奖励
- $N(v_i)$ = 该节点访问次数
- $c$ = 探索系数（通常1.4）

**检验标准**：MCTS(iterations=100) 应该 > 80% 胜率战胜规则AI
